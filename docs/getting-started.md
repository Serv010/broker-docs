# Getting Started

This guide walks through the normal user path from opening BRO-ker to making a first trade.

## 1. Open the Bot

Start BRO-ker in Telegram and send:

```text
/start
```

On first use, the bot may show terms and safety information. Read them carefully before continuing.

## 2. Set Up a Wallet

From the dashboard, choose the wallet setup action.

BRO-ker can generate a Solana wallet for your Telegram account. When a wallet is created, the bot shows backup material in a private chat. Save it somewhere secure before funding the wallet.

!!! danger "Never share wallet secrets"
    Never share your private key, seed phrase, Telegram login, or one-time codes with anyone. BRO-ker support will not ask for them.

## 3. Fund the Wallet

Send SOL to your BRO-ker wallet address. SOL is needed for:

- Buying tokens.
- Network fees.
- Priority fees or protected sending when enabled.
- Creating token accounts when required.

After funding, refresh the wallet or dashboard to confirm the balance.

## 4. Open a Token

Paste a valid Solana token mint into the chat. BRO-ker opens a token sheet with:

- Token name and symbol.
- Mint address.
- Price and market cap when available.
- Wallet balance and estimated value.
- Buy, sell, limit, chart, settings, and portfolio actions.

## 5. Review a Quote

Choose a preset buy amount, a preset sell amount, or enter a custom amount. BRO-ker fetches a live executable quote and displays the expected result, slippage, price impact warnings, route information, and quote expiration.

## 6. Confirm the Trade

If trade confirmation is enabled, review the quote and tap confirm. The bot submits the transaction and reports whether it was confirmed, sent but still pending, or failed.

## 7. Check Positions

Use the portfolio, token sheet, PnL card, or chart card to review tracked activity. PnL calculations are based on BRO-ker tracked trades and available live quote data.

## 8. Sell or Set Orders

From a token sheet, you can:

- Sell a preset percentage.
- Sell a custom amount.
- Use Sell Initial when available.
- Switch to Limit mode and create a buy or sell order.
- Open Limit Dash to review active orders.
